# Space Punk

> Proyecto de prácticas de FP — Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Web (DAW).

## ¿Qué es esto?

**Space Punk** es un juego web 2D top-down de mundo abierto con ambientación steampunk-fantasía. Es también un proyecto de aprendizaje para construir software desde cero aplicando prácticas profesionales:

- Definición y planificación de producto
- Stack moderno de desarrollo web (Phaser.js)
- Versionado con Git y trabajo con pull requests
- Testing y refactor
- Despliegue en producción
- Trabajo asistido por IA (copiloto y agentic)
